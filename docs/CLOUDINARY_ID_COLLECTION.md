## Quick Collection Method:

### Option A: Manual (5-10 minutes)
1. Open each folder in Cloudinary
2. Click on each video
3. Copy the Public ID (with the 6-character code)
4. Paste here

### Option B: Bulk Export (1 minute)
1. In Cloudinary, go to a folder
2. Select all videos (checkbox at top)
3. Click "Export" or look for a way to see all Public IDs at once
4. Copy the list
